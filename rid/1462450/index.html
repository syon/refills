Metalsmith & Wercker 自動デプロイ

2016.5.6

wercker.yml

box: node

build:
  steps:
    - npm-install

deploy:
  steps:
    - script:
        name: install git
        code: |
          apt-get update
          apt-get install git -y

    - script:
        name: Makefile make
        code: |
          make build

    - lukevivier/gh-pages:
        token: $GH_TOKEN
        basedir: refills

解説

  • box に git がインストールされていないためデプロイ時にインストール
  • lukevivier/gh-pages は lukevivier 氏が作った GitHub Pages デプロイ用の wercker step
    • lukevivier/step-gh-pages
    • lvivier/step-gh-pages: deploy github pages with wercker

GitHub アクセストークンと Wercker の設定

$GH_TOKEN は GitHub の個人設定で生成したトークンを Wercker 経由で割り当てています。 詳しくは別のページでまとめていますので参照ください。
