== JavaScript | Moment.js

|===
|Format |Sample

|`YYYYMMDD`
|`20200126`

|`YYYY-MM-DD HH:mm:ss`
|`2020-01-26 14:37:59`

|`YYYY/MM/DD HH:mm:ss.SSS`
|`2020/01/26 14:37:59.156`
|===

|`2020-01-26 14:37:59`
